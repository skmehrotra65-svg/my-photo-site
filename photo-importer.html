Photo Importer

Drop your curated photos in below. This reads the date, camera settings, and orientation straight from each file (nothing is uploaded anywhere — it all stays in your browser), and builds the code block your portfolio site needs.

1. Drop the exact photo files you want on the site (the ones you'll also copy into your images folder).
2. Fill in a title, location, category, and region for each one — everything else is auto-filled where possible.
3. Click Generate Code, then Copy.
4. Open data/photos.json, find the "photos": [ line, and paste the copied entries in as new items in that list (each one separated by a comma).
5. Copy the same original photo files into your site's images folder — filenames already match, so nothing else to rename.
6. If you're using the CMS at /admin, commit and push the updated data/photos.json and images folder to GitHub — the site rebuilds automatically. Otherwise, redeploy by dragging the folder into Netlify again.
